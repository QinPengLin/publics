砥砺奋进的五年——党的十八大以来我院国际合作交流工作

2018-08-28 01:38:23
近日，由创业时代网和京领教育共同发布的“中国大学国际化竞争力排行榜500强”名单公布，这是中国在建设“双一流”大学的背景下，国内首次从大学国际化的视角对大学竞争力进行综合排名，我院位居第446位，处于湖南省同类高校前列。这一成绩来之不易，是院党委、行政正确领导、谋划，全院师生员工共同努力奋斗的结果。是对我院国际合作交流工作的肯定。

回顾党的十八大以来的五年，是我院国际合作交流工作谋篇布局，开拓奋进的五年。五年来，面对高等教育国际化大趋势及我院国际合作交流工作“底子薄、基础差、起步晚”的局面，院党委要求高起点谋划和部署，全院上下根据“解放思想、稳步开局、顺势而为、全面出击、奋力追赶”的原则，广交天下朋友，“请进来”与“走出去”并重，在师资队伍国际化、学生交换、国外优质教育资源引进等方面取得了重大突破。

（一）党委高度重视，各种保障体制机制不断健全和完善

五年来，根据“党管外事”的原则，学院党委高度重视国际合作交流工作。一是成立了外事工作领导小组，对学院外事工作进行统一谋划和部署。二是健全工作机构。2012年，在学院机构调整设置中，学院党委将国际合作交流处从党政办公室分离，单独设置为学院职能工作部门，统一负责外事管理工作。2015年、2016年，又分别成立国际教育学院、港澳台事务办公室，与国际合作交流处合署办公。组织机构进一步健全与完善。三是外事工作经费得到充分保障。五年来，在学院财力较为紧张的情况下，涉外工作所需各项经费均得到充分保障，为国际合作交流工作的顺利开展提供了有力支撑。四是制度不断规范。《湖南工学院因公临时出国（境）实施意见》、《湖南工学院教职工因公出国留学管理办法》等一批涉外制度相继出台，推动国际合作交流工作不断走向规范。

（二）常规工作全面展开，平稳有序推进

五年来，按照“围绕中心、服务大局”的思路，结合学院发展实际，各项急需的涉外日常工作全面铺开，特别注重 “从无到有”的工作，即以前没有开展的工作，全面铺开推进，填补空白。同时，在确保工作向前大踏步推进的同时，统筹兼顾“平稳有序”，做到“忙而不急、杂而不乱”。五年来，共聘请外籍教师36人次来校工作，数量超过前10年的总和，工作效果良好，没有发生一起涉外事故；五年来，年均接待外宾（含港澳台地区）从2次到20余次，做到“热情而不超标、周到而不铺张”；五年来，聘请湖南省海外名师从1名到3名，全部顺利到校工作并圆满结题；五年来，教师获得国家留学基金委和省教育厅出国留学资助项目从1项增至16项，绝大部分已派出并学成归国；五年来，学校启动资助教职工出国留学项目，已选拔11人，全部顺利派出，其中9人学成归国；五年来，在校学生通过各种途径出国留学交流从2人次增至70余人次，实现手续办理“零投诉”；五年来，获批参加“千名中西部大学校长海外研修计划”项目从无到有，有四名院领导先后获批该项目，并全部学成归国，数量位居省内高校前列。

（三）重点工作取得历史性突破

在常规工作逐渐铺开的同时，为推动学院国际合作交流工作向“质量型、效益型”转变，五年来，不断创新思路、积极拓展，一些关键工作取得重大突破。

一是获批教育部中外合作办学项目。2014年，经过多办艰苦谈判与努力，与英国桑德兰大学联合举办的电气工程及其自动化本科教育项目获教育部批准，取得历史性突破。该项目于2015年开始招生，目前有在校学生230名，运转正常有序。这是湘南地区唯一正在运行的中外合作办学本科教育项目，也是我院国际合作交流工作发展史上的重大突破和对外交往的一张靓丽名片。

二是获批为来华留学生招生单位。为推动学院“走出去”发展战略的实施，在学院党委、行政的领导下，积极寻求上级政策支持，通过申报评审，学院于2014年获批为来华留学生招生单位，这是学院国际合作交流工作的发展的里程碑，学院将以此为抓手，推动学院从“单一引进”向“引进与输出”并重的发展。

三是境外办学筹备正式启动。根据《中共中央办公厅国务院办公厅〈关于做好新时期教育对外开放工作的若干意见〉的通知》、《国家发展改革委、外交部、商务部经国务院授权发布的〈推动共建丝绸之路经济带和21世纪海上丝绸之路的愿景与行动〉》、《教育部〈推进共建“一带一路”教育行动〉》精神，配合湖南企业在“一带一路”沿线投资兴业，学院启动了在老挝办学的筹备工作。2017年3月，党委书记谢国保同志率团赴老挝，实地考察了老挝办学政策与条件，并与老挝教育部官员就办学的有关具体事项进行了洽谈。

回顾五年来的工作，我院国际合作交流发展态势良好，取得了一定的成绩，未来五年，相信在院党委、行政的坚强领导下，借助国家对外开放的大好形势、随着学院“双一流”建设和申硕工程的推进，国际合作交流工作必将取得更大的成绩与发展。国际合作交流处（港澳台事务办公室、国际教育学院）
