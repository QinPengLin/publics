广州千山医疗器械科技有限公司来校开展合作交流

2018-08-28 01:40:49
0月19日，广州千山医疗器械科技有限公司总经理、我校82级安全工程专业优秀校友李一行3人来校开展交流合作洽谈。副校长刘俊学、招生与就业指导处处长张兴、教务处副处长李理、电气与信息工程学院院长姚胜兴、党委副书记陆秀令等参加洽谈会。

李愉快地回忆了在母校求学的美好时光，感恩母校的教育培养；对母校的快速发展表示由衷赞叹。

李介绍了公司的发展状况、产品种类及专利、研发水平、毕业生人才需求等情况；高度评价近年来母校输送的多批次毕业生具有专业基础理论扎实、实践能力强、敢创新、能奉献、脚踏实地等优秀品质。李?衷心期盼与母校在人才培养、教师实践平台、实习就业等方面开展深度合作。

刘俊学副校长高度赞扬李校友所获得的创业成绩，鼓励他继续务实创新，把企业做得更大更强更优。

刘俊学介绍了学校办学定位、人才培养、教师实践平台、实习实训、毕业生就业等情况。刘俊学指出，学校高度重视与优秀校友企业合作，不断创新应用型人才培养模式，不断提高人才培养质量。刘俊学要求双方在应用型专业设置、课程设计、协同创新、实习实训基地共建、教师实践及实验平台建设、毕业生就业等方面，有效整合优势资源，完善交流合作机制，丰富合作内涵，注重合作实效。
