学校召开《行政事业单位内部控制规范》调研座谈会

2018-08-28 01:23:39
为规范学校财务管理工作，保证单位经济活动合法合规、安全有效，10月19日上午，副校长陈世平在电信学院3309室主持召开落实《行政事业单位内部控制规范》调研座谈会，各二级学院（教学部）院长（主任）、财务处相关人员出席会议。会上，各二级学院院长（主任）分别对本部门财务工作内控制度实施过程中遇到的问题、难点做了详细汇报，并提出了相应的建议。财务处处长谭德凡、核算科科长李鸿梅对与会人员所提问题做了详细解答。

陈世平在会上强调，财务工作内部控制不仅是新时期加强财务管理的需要，也是学校科学管理、规范管理的需要，更是反腐倡廉建设的必然要求，各部门、各单位要严格预算管理，严格执行财经法律法规及学校的财务管理规章制度，做到“不乱花钱、做实事有钱花”。（财务处李丽）
